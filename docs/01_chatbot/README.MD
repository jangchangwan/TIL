1. 저장


```python
a = 30
b = '안녕'
```

-  `=` 은 저장을 해주는 기호이다.
  - 변수에 넣어준다 

- `==` 은 동일하다는 의미이다.

  - ex.

    - ```python
      a == 3
      # a는 3이 맞니?
      ```

- 저장 종류

  - `리스트 (List)` 

    - ```  python
      dust = [58,40,70]
      print(dust[1])
      ```

  - `딕셔너리 (dictionary)`

    - ``` python
      dust = {'영등포구': 58 , '강남구' : 40}
      print(dust['영등포구'])
      ```

      

2. 조건

   **boolean** 연산자

   | True(진실) | False(거짓) |
   | ---------- | ----------- |
   | 1          | 0           |

   `or / |` 는 하나만 `T ` 여도 결과값이 `T` 이다.
